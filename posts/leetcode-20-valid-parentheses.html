Today we solve LeetCode 20 - Valid Parentheses.

Source: https://leetcode.com/problems/valid-parentheses/

English

Problem Summary

Given a string consisting of ()[]{}, return whether brackets are valid: every opening bracket must be closed by the same type in correct order.

Key Insight

Use a stack to track unmatched opening brackets. When seeing a closing bracket, it must match the top of stack; otherwise invalid immediately.

Algorithm Steps

1) Create an empty stack.
2) Scan characters from left to right.
3) If char is opening bracket, push it.
4) If char is closing bracket: stack must be non-empty and top must match; then pop.
5) After scan, stack must be empty to be valid.

Complexity Analysis

Time: O(n).
Space: O(n) in worst case.

Common Pitfalls

- Forgetting to check empty stack before pop.
- Matching bracket type incorrectly.
- Returning true without checking leftover openings.

Reference Implementations (Java / Go / C++ / Python / JavaScript)

class Solution {
    public boolean isValid(String s) {
        char[] stack = new char[s.length()];
        int top = -1;

        for (char c : s.toCharArray()) {
            if (c == '(' || c == '[' || c == '{') {
                stack[++top] = c;
            } else {
                if (top < 0) return false;
                char open = stack[top--];
                if ((c == ')' && open != '(') ||
                    (c == ']' && open != '[') ||
                    (c == '}' && open != '{')) {
                    return false;
                }
            }
        }

        return top == -1;
    }
}

中文

题目概述

给定只包含 ()[]{} 的字符串，判断括号是否有效：左括号必须由同类型右括号按正确顺序闭合。

核心思路

用栈保存尚未匹配的左括号。遇到右括号时，必须与栈顶类型一致，否则立即返回无效。

算法步骤

1）初始化空栈。
2）从左到右遍历字符。
3）左括号入栈。
4）右括号时检查栈非空且栈顶可匹配，随后弹栈。
5）遍历结束后栈必须为空，才是有效字符串。

复杂度分析

时间复杂度：O(n)。
空间复杂度：O(n)（最坏情况全是左括号）。

常见陷阱

- 弹栈前没判断空栈。
- 括号类型匹配条件写错。
- 忘记最后判断栈是否为空。
