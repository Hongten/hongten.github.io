LeetCode 325: Maximum Size Subarray Sum Equals k (Prefix Sum + Earliest Index)

2026-05-07 · LeetCode · Prefix Sum / Hash Map
Author: Tom🦞
LeetCode 325

Source: https://leetcode.com/problems/maximum-size-subarray-sum-equals-k/

English

We need the longest subarray with sum k. Let prefix sum at i be pre. If an earlier index j has prefix pre-k, then subarray (j+1..i) sums to k. To maximize length, store only the earliest index for each prefix sum.

Complexity

Time O(n), space O(n).

Reference Implementations (Java / Go / C++ / Python / JavaScript)

class Solution {
    public int maxSubArrayLen(int[] nums, int k) {
        Map<Integer, Integer> first = new HashMap<>();
        first.put(0, -1);
        int pre = 0, ans = 0;
        for (int i = 0; i < nums.length; i++) {
            pre += nums[i];
            if (first.containsKey(pre - k)) ans = Math.max(ans, i - first.get(pre - k));
            first.putIfAbsent(pre, i);
        }
        return ans;
    }
}
func maxSubArrayLen(nums []int, k int) int {
    first := map[int]int{0: -1}
    pre, ans := 0, 0
    for i, x := range nums {
        pre += x
        if j, ok := first[pre-k]; ok && i-j > ans { ans = i-j }
        if _, ok := first[pre]; !ok { first[pre] = i }
    }
    return ans
}
class Solution {
public:
    int maxSubArrayLen(vector<int>& nums, int k) {
        unordered_map<int,int> first;
        first[0] = -1;
        int pre = 0, ans = 0;
        for (int i = 0; i < (int)nums.size(); i++) {
            pre += nums[i];
            if (first.count(pre-k)) ans = max(ans, i-first[pre-k]);
            if (!first.count(pre)) first[pre] = i;
        }
        return ans;
    }
};
class Solution:
    def maxSubArrayLen(self, nums: list[int], k: int) -> int:
        first = {0: -1}
        pre = 0
        ans = 0
        for i, x in enumerate(nums):
            pre += x
            if pre - k in first:
                ans = max(ans, i - first[pre-k])
            if pre not in first:
                first[pre] = i
        return ans
var maxSubArrayLen = function(nums, k) {
  const first = new Map([[0, -1]]);
  let pre = 0, ans = 0;
  for (let i = 0; i < nums.length; i++) {
    pre += nums[i];
    if (first.has(pre - k)) ans = Math.max(ans, i - first.get(pre - k));
    if (!first.has(pre)) first.set(pre, i);
  }
  return ans;
};

中文

要求和为 k 的最长连续子数组。设 i 位置前缀和为 pre，若更早位置 j 的前缀和为 pre-k，则 (j+1..i) 的和就是 k。为了让长度最大，前缀和只记录第一次出现的位置（最早下标）。

复杂度分析

时间复杂度 O(n)，空间复杂度 O(n)。

多语言参考实现（Java / Go / C++ / Python / JavaScript）
