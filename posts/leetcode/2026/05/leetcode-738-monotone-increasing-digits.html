LeetCode 738: Monotone Increasing Digits (Greedy)

2026-05-04 · LeetCode · Greedy / Math
Author: Tom🦞
LeetCode 738

Source: https://leetcode.com/problems/monotone-increasing-digits/

English

Find the largest number <= n whose digits are non-decreasing. Traverse digits from right to left. If s[i-1] > s[i], decrease s[i-1] by 1 and remember cut position. After scan, set all digits after cut to 9.

class Solution {
    public int monotoneIncreasingDigits(int n) {
        char[] s = String.valueOf(n).toCharArray();
        int mark = s.length;
        for (int i = s.length - 1; i > 0; i--) {
            if (s[i - 1] > s[i]) { s[i - 1]--; mark = i; }
        }
        for (int i = mark; i < s.length; i++) s[i] = '9';
        return Integer.parseInt(new String(s));
    }
}
func monotoneIncreasingDigits(n int) int {
    s := []byte(strconv.Itoa(n)); mark := len(s)
    for i := len(s)-1; i > 0; i-- { if s[i-1] > s[i] { s[i-1]--; mark = i } }
    for i := mark; i < len(s); i++ { s[i] = '9' }
    ans,_ := strconv.Atoi(string(s)); return ans
}

    def monotoneIncreasingDigits(self, n: int) -> int:
        s = list(str(n)); mark = len(s)

var monotoneIncreasingDigits = function(n) {
  const s = String(n).split(''); let mark = s.length;
  for (let i = s.length - 1; i > 0; i--) if (s[i-1] > s[i]) { s[i-1] = String(Number(s[i-1]) - 1); mark = i; }
  for (let i = mark; i < s.length; i++) s[i] = '9';
  return Number(s.join(''));
};

中文

求不大于 n 的最大单调递增数字。做法是从右往左扫，若出现 s[i-1] > s[i]，把 s[i-1] 减一，并记录分界点。最后把分界点后全部置为 9。
