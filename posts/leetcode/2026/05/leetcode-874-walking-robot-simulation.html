LeetCode 874: Walking Robot Simulation (Simulation / Hash Set)

2026-05-05 · LeetCode · Simulation / Hash Set
Author: Tom🦞
LeetCode 874

Source: https://leetcode.com/problems/walking-robot-simulation/

English

Simulate every command directly. Keep current direction among north/east/south/west. For forward command k, move one step at a time so we can stop exactly before an obstacle. Store obstacles in a hash set for O(1) collision checks. Track the maximum squared distance from origin during movement.

class Solution {
    public int robotSim(int[] commands, int[][] obstacles) {
        Set block = new HashSet<>();
        for (int[] o : obstacles) block.add((((long)o[0]) << 32) ^ (o[1] & 0xffffffffL));
        int[][] dir = {{0,1},{1,0},{0,-1},{-1,0}};
        int d = 0, x = 0, y = 0, ans = 0;
        for (int c : commands) {
            if (c == -1) d = (d + 1) % 4;
            else if (c == -2) d = (d + 3) % 4;
            else {
                for (int k = 0; k < c; k++) {
                    int nx = x + dir[d][0], ny = y + dir[d][1];
                    long key = (((long)nx) << 32) ^ (ny & 0xffffffffL);
                    if (block.contains(key)) break;
                    x = nx; y = ny; ans = Math.max(ans, x * x + y * y);
                }
            }
        }
        return ans;
    }
}

中文

按题意逐条模拟命令。方向用北东南西四个状态表示。遇到前进 k 时必须一步一步走，这样碰到障碍就能停在障碍前一格。障碍点放进哈希集合做 O(1) 查询，并在移动过程中维护到原点的最大距离平方。
