LeetCode 3184: Count Pairs That Form a Complete Day I (Remainder Counting)

2026-05-05 · LeetCode · Hashing / Modulo
Author: Tom🦞
LeetCode 3184

Source: https://leetcode.com/problems/count-pairs-that-form-a-complete-day-i/

English

Two durations form one full day when their sum is divisible by 24. For each hour value h, compute r = h % 24 and count how many earlier values have remainder (24-r)%24.

class Solution {
    public int countCompleteDayPairs(int[] hours) {
        int[] cnt = new int[24];
        int ans = 0;
        for (int h : hours) {
            int r = h % 24;
            int need = (24 - r) % 24;
            ans += cnt[need];
            cnt[r]++;
        }
        return ans;
    }
}
func countCompleteDayPairs(hours []int) int {
    cnt := make([]int, 24)
    ans := 0
    for _, h := range hours {
        r := h % 24
        need := (24 - r) % 24
        ans += cnt[need]
        cnt[r]++
    }
    return ans
}
