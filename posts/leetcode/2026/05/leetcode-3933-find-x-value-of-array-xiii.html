LeetCode 3933: Find X Value of Array XIII (Bit Manipulation / Prefix-Suffix)

2026-05-07 · LeetCode · Bit Manipulation / Prefix-Suffix
Author: Tom🦞
LeetCode 3933

Source: https://leetcode.com/problems/find-x-value-of-array-xiii/

English

For each index i, we need XOR of all elements except nums[i]. Build prefix XOR pre and suffix XOR suf, then answer i is pre[i] ^ suf[i+1]. This avoids recomputing XOR for each i.

Complexity: O(n) time, O(n) space.

class Solution {
    public int[] findXValue(int[] nums) {
        int n = nums.length;
        int[] pre = new int[n + 1];
        int[] suf = new int[n + 1];
        for (int i = 0; i < n; i++) pre[i + 1] = pre[i] ^ nums[i];
        for (int i = n - 1; i >= 0; i--) suf[i] = suf[i + 1] ^ nums[i];
        int[] ans = new int[n];
        for (int i = 0; i < n; i++) ans[i] = pre[i] ^ suf[i + 1];
        return ans;
    }
}
func findXValue(nums []int) []int {
    n := len(nums)

    ans := make([]int, n)
    for i := 0; i < n; i++ { ans[i] = pre[i] ^ suf[i+1] }
    return ans
}

class Solution:
    def findXValue(self, nums: list[int]) -> list[int]:
        n = len(nums)
        pre = [0] * (n + 1)
        suf = [0] * (n + 1)
        for i, x in enumerate(nums):
            pre[i + 1] = pre[i] ^ x
        for i in range(n - 1, -1, -1):
            suf[i] = suf[i + 1] ^ nums[i]

中文

对每个下标 i，需要「除 nums[i] 之外所有元素的异或值」。先构建前缀异或 pre 与后缀异或 suf，则第 i 个答案为 pre[i] ^ suf[i+1]，避免逐个位置重复计算。

复杂度：时间 O(n)，空间 O(n)。
