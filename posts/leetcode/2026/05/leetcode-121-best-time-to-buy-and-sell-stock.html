Today we solve LeetCode 121 - Best Time to Buy and Sell Stock.

Source: https://leetcode.com/problems/best-time-to-buy-and-sell-stock/

English

Problem Summary

Given prices[i], pick one buy day and one later sell day to maximize profit. Return 0 if no profit is possible.

Key Insight

While scanning from left to right, keep the minimum price seen so far. At each day, best sell profit is current price minus that minimum.

Complexity Analysis

Time: O(n).
Space: O(1).

Reference Implementations (Java / Go / C++ / Python / JavaScript)

class Solution { public int maxProfit(int[] prices) { int minPrice = Integer.MAX_VALUE, ans = 0; for (int p : prices) { minPrice = Math.min(minPrice, p); ans = Math.max(ans, p - minPrice); } return ans; } }
func maxProfit(prices []int) int { minPrice, ans := 1<<30, 0; for _, p := range prices { if p < minPrice { minPrice = p }; if p-minPrice > ans { ans = p-minPrice } }; return ans }
class Solution { public: int maxProfit(vector<int>& prices) { int minPrice = INT_MAX, ans = 0; for (int p : prices) { minPrice = min(minPrice, p); ans = max(ans, p - minPrice); } return ans; } };
class Solution:
    def maxProfit(self, prices: list[int]) -> int:
        min_price = float('inf')
        ans = 0
        for p in prices:
            min_price = min(min_price, p)
            ans = max(ans, p - min_price)
        return ans
var maxProfit = function(prices) { let minPrice = Infinity, ans = 0; for (const p of prices) { minPrice = Math.min(minPrice, p); ans = Math.max(ans, p - minPrice); } return ans; };

中文

题目概述

给定 prices[i]，只能买一次、卖一次（且卖出日在买入日之后），求最大利润；若无法获利返回 0。

核心思路

从左到右遍历，维护截至当前的最低买入价 minPrice。把今天作为卖出日时利润是 p-minPrice，持续更新最大值即可。

复杂度分析

时间复杂度：O(n)。
空间复杂度：O(1)。
