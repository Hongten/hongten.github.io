LeetCode 2418: Sort the People (Pair-and-Sort by Height Descending)

Today we solve LeetCode 2418 - Sort the People.

Source: https://leetcode.com/problems/sort-the-people/

English

Problem Summary

You are given two arrays: names[i] and heights[i] belong to the same person. Return names sorted by height in descending order.

Key Insight

The person identity is the pair (name, height). Once we keep each pair together, the task becomes a direct descending sort by height.

Algorithm

1) Build pairs from the two arrays.
2) Sort pairs by height descending.
3) Extract names in sorted order.

Complexity Analysis

Time: O(n log n) because of sorting.
Space: O(n) for paired records / output.

Common Pitfalls

- Sorting heights alone and losing which name each height belongs to.
- Sorting in ascending order by accident.
- Repeatedly searching max height in loops, which can degrade performance.

Reference Implementations (Java / Go / C++ / Python / JavaScript)

class Solution {
    public String[] sortPeople(String[] names, int[] heights) {
        int n = names.length;
        Integer[] idx = new Integer[n];
        for (int i = 0; i < n; i++) idx[i] = i;

        java.util.Arrays.sort(idx, (a, b) -> Integer.compare(heights[b], heights[a]));

        String[] ans = new String[n];
        for (int i = 0; i < n; i++) {
            ans[i] = names[idx[i]];
        }
        return ans;
    }
}
func sortPeople(names []string, heights []int) []string {
    n := len(names)
    idx := make([]int, n)
    for i := 0; i < n; i++ {
        idx[i] = i
    }

    sort.Slice(idx, func(i, j int) bool {
        return heights[idx[i]] > heights[idx[j]]
    })

    ans := make([]string, n)
    for i := 0; i < n; i++ {
        ans[i] = names[idx[i]]
    }
    return ans
}
class Solution {
public:
    vector<string> sortPeople(vector<string>& names, vector<int>& heights) {
        int n = names.size();
        vector<int> idx(n);
        iota(idx.begin(), idx.end(), 0);

        sort(idx.begin(), idx.end(), [&](int a, int b) {
            return heights[a] > heights[b];
        });

        vector<string> ans;
        ans.reserve(n);
        for (int i : idx) ans.push_back(names[i]);
        return ans;
    }
};
class Solution:
    def sortPeople(self, names: list[str], heights: list[int]) -> list[str]:
        order = sorted(range(len(names)), key=lambda i: -heights[i])
        return [names[i] for i in order]
var sortPeople = function(names, heights) {
  const idx = Array.from({ length: names.length }, (_, i) => i);
  idx.sort((a, b) => heights[b] - heights[a]);
  return idx.map(i => names[i]);
};

中文

题目概述

给定两个数组，names[i] 与 heights[i] 表示同一个人。请按身高降序返回姓名数组。

核心思路

关键是先把“人”作为整体处理（姓名与身高绑定），再按身高排序。不要把 heights 单独排完后再去猜对应姓名。

算法步骤

1）建立下标数组或姓名-身高配对结构。
2）按身高从大到小排序。
3）按排序结果提取姓名。

复杂度分析

时间复杂度：O(n log n)（排序主导）。
空间复杂度：O(n)（存储下标/配对与结果）。

常见陷阱

- 只排序身高导致姓名映射丢失。
- 排序方向写反（升序而非降序）。
- 用双重循环反复找最大值，复杂度变高且代码更易错。
