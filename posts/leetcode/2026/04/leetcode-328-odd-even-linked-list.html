LeetCode 328: Odd Even Linked List (In-Place Two-Chain Stitching)

Today we solve LeetCode 328 - Odd Even Linked List.

Source: https://leetcode.com/problems/odd-even-linked-list/

English

Problem Summary

Given the head of a singly linked list, group all nodes at odd indices together followed by all nodes at even indices. The relative order inside odd and even groups must stay the same.

Key Insight

Maintain two running tails: one for odd-index nodes and one for even-index nodes. Rewire pointers while traversing, then connect odd tail to even head.

Algorithm

- If the list has fewer than 2 nodes, return head.
- Let odd = head, even = head.next, and keep evenHead = even.
- While even != null and even.next != null:
  1) odd.next = even.next, move odd.
  2) even.next = odd.next, move even.
- Finally, odd.next = evenHead.

Complexity Analysis

Time: O(n).
Space: O(1).

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode oddEvenList(ListNode head) {
        if (head == null || head.next == null) return head;

        ListNode odd = head;
        ListNode even = head.next;
        ListNode evenHead = even;

        while (even != null && even.next != null) {
            odd.next = even.next;
            odd = odd.next;

            even.next = odd.next;
            even = even.next;
        }

        odd.next = evenHead;
        return head;
    }
}

/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var oddEvenList = function(head) {
  if (head === null || head.next === null) return head;

  let odd = head;
  let even = head.next;
  const evenHead = even;

  while (even !== null && even.next !== null) {
    odd.next = even.next;
    odd = odd.next;

    even.next = odd.next;
    even = even.next;
  }

  odd.next = evenHead;
  return head;
};

中文

题目概述

给定单链表头节点，将奇数下标节点放在前面、偶数下标节点放在后面，并保持各自组内的相对顺序不变。

核心思路

用两个指针链分别维护奇链和偶链的尾部，遍历过程中原地改指针，最后把奇链尾连接到偶链头。

算法步骤

- 当链表长度小于 2，直接返回。
- 设 odd = head，even = head.next，保存 evenHead = even。
- 循环条件为 even != null 且 even.next != null：
  1）odd.next = even.next，odd 前进。
  2）even.next = odd.next，even 前进。
- 循环结束后执行 odd.next = evenHead。

复杂度分析

时间复杂度：O(n)。
空间复杂度：O(1)。
