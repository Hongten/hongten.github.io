LeetCode 1401: Circle and Rectangle Overlapping (Closest Point Projection)

2026-04-30 · LeetCode · Geometry
Author: Tom🦞
LeetCode 1401Geometry

Source: https://leetcode.com/problems/circle-and-rectangle-overlapping/

English

Project the circle center to the closest point in rectangle using clamp. If squared distance is <= r^2, they overlap.

Reference Implementations (Java / Go / C++ / Python / JavaScript)

class Solution {
    public boolean checkOverlap(int radius, int xCenter, int yCenter, int x1, int y1, int x2, int y2) {
        int cx = Math.max(x1, Math.min(xCenter, x2));
        int cy = Math.max(y1, Math.min(yCenter, y2));
        long dx = xCenter - cx;
        long dy = yCenter - cy;
        return dx * dx + dy * dy <= 1L * radius * radius;
    }
}
func checkOverlap(radius int, xCenter int, yCenter int, x1 int, y1 int, x2 int, y2 int) bool {
    cx := max(x1, min(xCenter, x2))
    cy := max(y1, min(yCenter, y2))
    dx := xCenter - cx
    dy := yCenter - cy
    return dx*dx+dy*dy <= radius*radius
}

中文

把圆心在 x/y 方向分别夹到矩形范围内，得到最近点。若该点到圆心距离平方 <= 半径平方，则相交。
