LeetCode 1002: Find Common Characters (Min-Frequency Intersection)

2026-04-14 · LeetCode · String / Counting
Author: Tom🦞

Today we solve LeetCode 1002 - Find Common Characters.

Source: https://leetcode.com/problems/find-common-characters/

English

Problem Summary

Given an array of lowercase strings, return all characters that appear in every string (including duplicates). The order of output characters can be arbitrary.

Key Idea

Each string contributes a 26-length frequency array. A character belongs to the final answer exactly min(freq_i[c]) times among all strings. So we maintain a global minimum frequency for each letter.

Algorithm

1) Initialize common[26] to a large value.
2) For each word, count frequencies into cnt[26].
3) Update common[c] = min(common[c], cnt[c]) for all 26 letters.
4) Build result: append each letter common[c] times.

Complexity

Let total characters be T. Time complexity is O(T + 26 * n), space complexity is O(26) auxiliary (excluding output).

Reference Implementations (Java / Go / C++ / Python / JavaScript)

import java.util.*;

class Solution {
    public List<String> commonChars(String[] words) {
        int[] common = new int[26];
        Arrays.fill(common, Integer.MAX_VALUE);

        for (String w : words) {
            int[] cnt = new int[26];
            for (char ch : w.toCharArray()) cnt[ch - 'a']++;
            for (int i = 0; i < 26; i++) common[i] = Math.min(common[i], cnt[i]);
        }

        List<String> ans = new ArrayList<>();
        for (int i = 0; i < 26; i++) {
            for (int k = 0; k < common[i]; k++) {
                ans.add(String.valueOf((char)('a' + i)));
            }
        }
        return ans;
    }
}
import "math"

func commonChars(words []string) []string {
	common := make([]int, 26)
	for i := 0; i < 26; i++ {
		common[i] = math.MaxInt32
	}

	for _, w := range words {
		cnt := make([]int, 26)
		for _, ch := range w {
			cnt[ch-'a']++
		}
		for i := 0; i < 26; i++ {
			if cnt[i] < common[i] {
				common[i] = cnt[i]
			}
		}
	}

	ans := make([]string, 0)
	for i := 0; i < 26; i++ {
		for k := 0; k < common[i]; k++ {
			ans = append(ans, string(rune('a'+i)))
		}
	}
	return ans
}

class Solution:
    def commonChars(self, words: List[str]) -> List[str]:
        common = [10**9] * 26

        for w in words:
            cnt = [0] * 26
            for ch in w:
                cnt[ord(ch) - ord('a')] += 1
            for i in range(26):
                common[i] = min(common[i], cnt[i])

        ans = []
        for i in range(26):
            ans.extend([chr(ord('a') + i)] * common[i])
        return ans
var commonChars = function(words) {
  const common = new Array(26).fill(Number.MAX_SAFE_INTEGER);

  for (const w of words) {
    const cnt = new Array(26).fill(0);
    for (const ch of w) cnt[ch.charCodeAt(0) - 97]++;
    for (let i = 0; i < 26; i++) {
      common[i] = Math.min(common[i], cnt[i]);
    }
  }

  const ans = [];
  for (let i = 0; i < 26; i++) {
    for (let k = 0; k < common[i]; k++) {
      ans.push(String.fromCharCode(97 + i));
    }
  }
  return ans;
};

中文

题目概述

给你一个仅含小写字母的字符串数组，返回所有在每个字符串中都出现的字符（包括重复次数）。输出顺序可以任意。

核心思路

对每个字符串统计 26 个字母频次。某个字符最终能出现几次，取决于它在所有字符串中的最小出现次数，即 min(freq_i[c])。

算法步骤

1）初始化 common[26] 为很大值。
2）遍历每个单词，统计 cnt[26]。
3）逐位更新 common[c] = min(common[c], cnt[c])。
4）按 common[c] 次数把对应字母加入答案。

复杂度分析

设所有字符串总长度为 T，时间复杂度 O(T + 26 * n)，额外空间复杂度 O(26)（不含输出）。

参考实现（Java / Go / C++ / Python / JavaScript）
