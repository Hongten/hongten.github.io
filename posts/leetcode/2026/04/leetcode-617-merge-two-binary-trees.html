Today we solve LeetCode 617 - Merge Two Binary Trees.

Source: https://leetcode.com/problems/merge-two-binary-trees/

English

Problem Summary

Given two binary trees, merge them into one tree: if both nodes exist, new value is their sum; otherwise use the non-null node.

Key Insight

The merge rule is local and identical for every pair of aligned nodes, so DFS recursion fits naturally: merge root, then merge left children and right children.

Algorithm

- If root1 is null, return root2.
- If root2 is null, return root1.
- Both exist: add values into root1.val.
- Recursively set root1.left = mergeTrees(root1.left, root2.left).
- Recursively set root1.right = mergeTrees(root1.right, root2.right).
- Return root1.

Complexity Analysis

Time: O(n), where n is the number of visited nodes in the union of both trees.
Space: O(h) recursion stack, h is tree height.

Common Pitfalls

- Forgetting to return the non-null subtree when the other side is null.
- Creating unnecessary new nodes when in-place merge is allowed.
- Confusing preorder/inorder/postorder; any works, but recursion must merge both children consistently.

Reference Implementations (Java / Go / C++ / Python / JavaScript)

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public TreeNode mergeTrees(TreeNode root1, TreeNode root2) {
        if (root1 == null) return root2;
        if (root2 == null) return root1;

        root1.val += root2.val;
        root1.left = mergeTrees(root1.left, root2.left);
        root1.right = mergeTrees(root1.right, root2.right);
        return root1;
    }
}
/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */
func mergeTrees(root1 *TreeNode, root2 *TreeNode) *TreeNode {
    if root1 == nil {
        return root2
    }
    if root2 == nil {
        return root1
    }

    root1.Val += root2.Val
    root1.Left = mergeTrees(root1.Left, root2.Left)
    root1.Right = mergeTrees(root1.Right, root2.Right)
    return root1
}

 * Definition for a binary tree node.

 *     this.val = (val===undefined ? 0 : val)

 *     this.right = (right===undefined ? null : right)
 * }

/**
 * @param {TreeNode} root1
 * @param {TreeNode} root2

var mergeTrees = function(root1, root2) {
  if (root1 === null) return root2;
  if (root2 === null) return root1;

  root1.left = mergeTrees(root1.left, root2.left);
  root1.right = mergeTrees(root1.right, root2.right);
  return root1;

中文

题目概述

给定两棵二叉树，按规则合并：若两个位置都有节点，则新值为两者之和；若只有一边有节点，则直接使用非空节点。

核心思路

合并规则对每一对对齐节点都一样，天然适合递归 DFS：先处理当前节点，再分别合并左子树和右子树。

算法步骤

- 若 root1 为空，返回 root2。
- 若 root2 为空，返回 root1。
- 两者都非空时，执行 root1.val += root2.val。
- 递归合并左子树与右子树并回写到 root1。
- 返回 root1 作为合并结果。

复杂度分析

时间复杂度：O(n)，n 为两棵树并集里被访问的节点数。
空间复杂度：O(h)，主要是递归栈深度，h 为树高。

常见陷阱

- 忘记在一侧为空时直接返回另一侧。
- 题目允许原地修改时却额外创建大量新节点。
- 只合并了一边子树，导致结构不完整。
