Today we solve LeetCode 997 - Find the Town Judge.

Source: https://leetcode.com/problems/find-the-town-judge/

English

Problem Summary

In a town of n people labeled from 1 to n, the judge is trusted by everyone else but trusts nobody. Given trust pairs [a, b], return the judge label or -1 if it does not exist.

Key Insight

Judge conditions can be turned into degree conditions in a directed graph: judge has inDegree = n - 1 and outDegree = 0. We can merge both into one score: degree[i] = inDegree[i] - outDegree[i]. The judge is exactly the node with score n - 1.

Algorithm

- Initialize degree array size n + 1.
- For each trust edge a -> b: do degree[a]-- and degree[b]++.
- Scan people 1..n; if degree[i] == n - 1, return i.
- If none matches, return -1.

Complexity Analysis

Let m be number of trust pairs.
Time: O(n + m).
Space: O(n).

Common Pitfalls

- Confusing "trusted by all others" with "trusted by at least one".
- Forgetting that candidate must trust nobody.
- Not handling n = 1 (answer is 1 when no constraints violate).

Reference Implementations (Java / Go / C++ / Python / JavaScript)

class Solution {
    public int findJudge(int n, int[][] trust) {
        int[] degree = new int[n + 1];
        for (int[] edge : trust) {
            int a = edge[0], b = edge[1];
            degree[a]--; // out-degree contribution
            degree[b]++; // in-degree contribution
        }
        for (int person = 1; person <= n; person++) {
            if (degree[person] == n - 1) {
                return person;
            }
        }
        return -1;
    }
}

  }

};

中文

题目概述

小镇有 n 个人（编号 1..n）。法官满足两个条件：被其他所有人信任、且自己不信任任何人。给定信任关系 [a, b]，返回法官编号；若不存在返回 -1。

核心思路

把问题转成有向图的入度/出度判定：法官应满足 入度 = n - 1 且 出度 = 0。可合并为一个分值 degree[i] = 入度 - 出度，法官就是分值为 n - 1 的节点。

算法步骤

- 创建长度为 n + 1 的 degree 数组。
- 遍历每条关系 a -> b：执行 degree[a]--、degree[b]++。
- 遍历 1..n，若某人满足 degree[i] == n - 1，直接返回该编号。
- 若遍历结束仍无结果，返回 -1。

复杂度分析

设信任关系条数为 m。
时间复杂度：O(n + m)。
空间复杂度：O(n)。

常见陷阱

- 把“被所有其他人信任”误解成“被至少一个人信任”。
- 忽略“法官不能信任任何人”的限制。
- 忘记处理 n = 1 的边界情况。
