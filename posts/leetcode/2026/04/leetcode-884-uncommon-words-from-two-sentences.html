LeetCode 884: Uncommon Words from Two Sentences (Frequency Map)

Today we solve LeetCode 884 - Uncommon Words from Two Sentences.

Source: https://leetcode.com/problems/uncommon-words-from-two-sentences/

English

Problem Summary

Given two sentences s1 and s2, return all words that appear exactly once in one sentence and do not appear in the other sentence.

Key Insight

If we count all word frequencies across both sentences, uncommon words are exactly the words with total frequency 1.

Brute Force and Limitations

A naive way is to compare each word with all others repeatedly, which is inefficient and hard to maintain. A hash map gives direct counting in linear time.

Optimal Algorithm Steps

1) Split both sentences by spaces into words.
2) Count each word in a hash map.
3) Collect all words whose count is 1.

Complexity Analysis

Let n be the total number of words in both sentences.
Time: O(n).
Space: O(n).

Common Pitfalls

- Returning words that appear once in each sentence (total 2), which is not uncommon.
- Forgetting to process both sentences into the same frequency table.

Reference Implementations (Java / Go / C++ / Python / JavaScript)

class Solution {
    public String[] uncommonFromSentences(String s1, String s2) {
        Map<String, Integer> freq = new HashMap<>();

        for (String w : s1.split(" ")) {
            freq.put(w, freq.getOrDefault(w, 0) + 1);
        }
        for (String w : s2.split(" ")) {
            freq.put(w, freq.getOrDefault(w, 0) + 1);
        }

        List<String> ans = new ArrayList<>();
        for (Map.Entry<String, Integer> e : freq.entrySet()) {
            if (e.getValue() == 1) {
                ans.add(e.getKey());
            }
        }

        return ans.toArray(new String[0]);
    }
}
func uncommonFromSentences(s1 string, s2 string) []string {
    freq := map[string]int{}

    for _, w := range strings.Split(s1, " ") {
        freq[w]++
    }
    for _, w := range strings.Split(s2, " ") {
        freq[w]++
    }

    ans := make([]string, 0)
    for w, c := range freq {
        if c == 1 {
            ans = append(ans, w)
        }
    }

    return ans
}
class Solution {
public:
    vector<string> uncommonFromSentences(string s1, string s2) {
        unordered_map<string, int> freq;
        for (const string& w : split(s1)) freq[w]++;
        for (const string& w : split(s2)) freq[w]++;

        vector<string> ans;
        for (auto& [w, c] : freq) {
            if (c == 1) ans.push_back(w);
        }
        return ans;
    }

private:
    vector<string> split(const string& s) {
        vector<string> out;
        string cur;
        for (char ch : s) {
            if (ch == ' ') {
                if (!cur.empty()) out.push_back(cur);
                cur.clear();
            } else {
                cur.push_back(ch);
            }
        }
        if (!cur.empty()) out.push_back(cur);
        return out;
    }
};
class Solution:
    def uncommonFromSentences(self, s1: str, s2: str) -> list[str]:
        freq: dict[str, int] = {}

        for w in s1.split():
            freq[w] = freq.get(w, 0) + 1
        for w in s2.split():
            freq[w] = freq.get(w, 0) + 1

        return [w for w, c in freq.items() if c == 1]
/**
 * @param {string} s1
 * @param {string} s2
 * @return {string[]}
 */
var uncommonFromSentences = function(s1, s2) {
  const freq = new Map();

  for (const w of s1.split(" ")) {
    freq.set(w, (freq.get(w) || 0) + 1);
  }
  for (const w of s2.split(" ")) {
    freq.set(w, (freq.get(w) || 0) + 1);
  }

  const ans = [];
  for (const [w, c] of freq.entries()) {
    if (c === 1) ans.push(w);
  }
  return ans;
};

中文

题目概述

给定两个句子 s1 与 s2，找出所有“非常见单词”：它只在某一个句子中出现一次，且在另一个句子中不出现。

核心思路

把两个句子的单词放到同一个频次表里统计。最终频次等于 1 的单词，就是答案。

暴力解法与不足

若反复两两比较单词会重复工作且代码复杂。哈希表计数更直接，时间复杂度线性。

最优算法步骤

1）按空格切分两个句子。
2）使用哈希表累计每个单词出现次数。
3）筛出频次为 1 的单词。

复杂度分析

设两个句子的总单词数为 n。
时间复杂度：O(n)。
空间复杂度：O(n)。

常见陷阱

- 把“两个句子各出现一次”的单词误判为答案（总频次是 2）。
- 没有把两个句子合并到同一张频次表统计。
