LeetCode 42: Trapping Rain Water (Two Pointers)

Today we solve LeetCode 42 - Trapping Rain Water.

English

Idea: Water at index i is limited by min(maxLeft, maxRight) - height[i]. Use two pointers and maintain running left/right max in one pass.

Complexity: O(n) time, O(1) extra space.

class Solution {
    public int trap(int[] h) {
        int l = 0, r = h.length - 1, lMax = 0, rMax = 0, ans = 0;
        while (l < r) {
            if (h[l] < h[r]) {
                lMax = Math.max(lMax, h[l]);
                ans += lMax - h[l++];
            } else {
                rMax = Math.max(rMax, h[r]);
                ans += rMax - h[r--];
            }
        }
        return ans;
    }
}
func trap(h []int) int {
    l, r := 0, len(h)-1
    lMax, rMax, ans := 0, 0, 0
    for l < r {
        if h[l] < h[r] {
            if h[l] > lMax { lMax = h[l] }
            ans += lMax - h[l]
            l++
        } else {
            if h[r] > rMax { rMax = h[r] }
            ans += rMax - h[r]
            r--
        }
    }
    return ans
}
class Solution {
public:
    int trap(vector<int>& h) {
        int l=0,r=(int)h.size()-1,lm=0,rm=0,ans=0;
        while(l<r){
            if(h[l]<h[r]){ lm=max(lm,h[l]); ans+=lm-h[l++]; }
            else { rm=max(rm,h[r]); ans+=rm-h[r--]; }
        }
        return ans;
    }
};
class Solution:
    def trap(self, h: List[int]) -> int:
        l, r = 0, len(h)-1
        lmax = rmax = ans = 0

                ans += lmax - h[l]
                l += 1
            else:
                rmax = max(rmax, h[r])
                ans += rmax - h[r]
                r -= 1
        return ans

中文

思路：第 i 列可接水量是 min(左侧最高, 右侧最高) - 当前高度。双指针从两端向中间收缩，动态维护左右最高值即可一趟算完。

复杂度：时间 O(n)，额外空间 O(1)。
